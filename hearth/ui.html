Auto-boot last model on launch

            
When you start Hearth, automatically load the model you used last. Saves the 15-30s "wait for me to click Use this" beat.

          

          

        


        

          

            
Default context window

            
Used for newly loaded models that don't have a saved config yet. Higher = more memory headroom but more KV cache VRAM.

          

          
        


        

          

            
Idle auto-unload

            
How long the server can sit idle before unloading the model to free VRAM. 0 = never unload. Same as HEARTH_KEEP_ALIVE_MIN.

          

          
        


        

          

            
Server port

            
Default 1234 to match LM Studio. Change only if you have a conflict.

          

          
        


        

          Port / idle changes take effect on the next server start. Restart the server above to apply now.
